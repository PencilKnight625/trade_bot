# Binance Futures Testnet Trading Bot

## Objective
This project is a simplified trading bot built using Binance Futures Testnet (USDT-M). It allows users to place market and limit orders, fetch account balance, 
and view open positions through a simple command-line interface.

## Features
- Place Market Orders (Buy/Sell)
- Place Limit Orders
- Fetch Futures Account Balance
- View Open Positions
- Check Order Status
- Simple CLI-based interaction

## Tech Stack
- Python 3.x
- Binance Futures API
- requests / python-binance (depending on implementation)

bash
git clone https://github.com/PencilKnight625/trade_bot.git
cd trade_bot
pip install -r requirements.txt
python main.py
